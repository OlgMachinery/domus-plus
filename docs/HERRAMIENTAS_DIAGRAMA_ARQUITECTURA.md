# Herramientas del diagrama de arquitectura

Lista para marcar **Sí** (mantener/añadir) o **No** (quitar/no usar). Cuando decidas, dímelo y ajustamos el código.

---

## Barra principal

| # | Herramienta | Descripción | ¿Incluir? (Sí/No) |
|---|-------------|-------------|-------------------|
| 1 | Versión correcta (prueba) | Pill verde "✓ Versión correcta (prueba)" | |
| 2 | Tabs: Jerarquía / Responsables / Presupuesto | Cambian la vista del diagrama | |
| 3 | Selector de layout | Desplegable: Vertical, Horizontal, Compacto, Radial, Layered, Force | |
| 4 | Auto ajustar | Ajusta zoom/pan para que todo el diagrama quepa | |
| 5 | Recalcular layout | Vuelve a calcular posiciones con el algoritmo actual | |
| 6 | Reset layout | Sale del modo personalizado, borra posiciones guardadas | |
| 7 | Descargar imagen | Exporta el canvas como PNG | |
| 8 | Más / Menos | Muestra u oculta la segunda fila de controles (sobre todo móvil) | |
| 9 | Cerrar | Botón rojo que vuelve a `/ui` | |
| 10 | Entidad + Volver | Si hay entidad enfocada: nombre y botón "Volver" | |
